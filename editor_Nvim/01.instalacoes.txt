instalar_plugin_que_instala_plugins:
- instalar plugins com  Vundle :
- defina a pasta que ficaram salvos os plugins
- clone o repo do Vundle para esta pasta
- repoVundle: https://github.com/VundleVim/Vundle.vim

- funcao de instalarPlugins aponte para buscar a pasta onde esta os
ex: vim.call('plug#begin', '~/.config/nvim_plugs_instalados/NvimRZJ235/')

  Plug 'dracula/vim'

vim.call('plug#end')
